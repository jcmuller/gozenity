## Install

```
go get github.com/jcmuller/gozenity
```

## Examples

See [godoc](https://godoc.org/github.com/jcmuller/gozenity#example-List)

### Simple input
```go
input, err := gozenity.Entry("Enter something:", "placeholder value")

if err != nil {
    log.Panic("Something happened: ", err)
}

fmt.Println(input)
```
